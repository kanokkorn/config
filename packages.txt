Packages
--------

list of essential packages and libraries for build a project from scratch.

on fresh install system, update package manager is recommended

Debian based
------------

based on debian-testing, stable channel or some distro might not have it list in package manager

- baseline tools

  sudo gcc make curl git

- bspwm (tiling window manager)

  libxcb-xinerama0-dev libxcb-icccm4-dev libxcb-randr0-dev libxcb-util0-dev libxcb-ewmh-dev\
  libxcb-keysyms1-dev libxcb-shape0-dev

- sxhkd (X event input handler)

  libxcb-keysyms1-dev

- rofi (window switcher, apps launcher)

  meson flex bison
